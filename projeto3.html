Missão Scratch Python JavaScript

Você já se perguntou como os jogos online, aplicativos e sites que você usa todos os dias funcionam? Por trás de tudo isso, existem linguagens de programação, que são como as instruções que os computadores seguem para fazer as coisas acontecerem na tela. Uma dessas linguagens muito importantes na programação é o JavaScript. Com ela, você pode criar coisas incríveis e interativas na internet, como jogos, animações e muito mais. Por isso, vamos juntos descobrir o que é o JavaScript e como ele funciona, para que logo logo você possa usar com maestria tudo o que ela tem a oferecer para os seus futuros projetos.

O que é JavaScript?

JavaScript é uma linguagem de programação super divertida e poderosa, usada para dar vida às páginas da internet! Com ela, você pode fazer sites interativos, animados e muito legais. Sabe quando você clica em um botão e algo acontece na página? Ou quando um jogo online responde aos seus comandos? Muitas dessas coisas acontecem por causa do JavaScript.

Onde o JavaScript é usado?

  • Sites interativos: JavaScript ajuda a criar efeitos legais em sites, como mudar cores, exibir mensagens ou até fazer animações!
  • Jogos na web: Dá para criar jogos que funcionam diretamente no navegador, sem precisar baixar nada.
  • Aplicativos da Web: Programas como calculadoras e calendários online são feitos com JavaScript.
  • Robôs e gadgets: Sabia que alguns robôs e dispositivos inteligentes também podem ser programados usando JavaScript?

O JavaScript é como uma ferramenta mágica que os programadores usam para deixar a internet mais divertida e interativa. E o melhor de tudo: você também pode aprender a usar dessa magia!

                     fetch('https://pokeapi.co/api/v2/pokemon?limit=10')

Passo 1: Configurar o ambiente

Para este projeto simples, você só precisará de um navegador e um editor de texto (como o Notepad, VSCode ou qualquer outro de sua escolha). Não é necessário instalar nada adicional, pois o JavaScript já é compatível com todos os navegadores modernos.

Passo 2: Criar o arquivo HTML

Vamos começar criando um arquivo HTML que servirá de estrutura para o contador:

1. Abra o editor de texto.
2. Crie um arquivo novo e salve-o como "contador.html".

Dentro desse arquivo, adicione o seguinte código HTML básico: 

    
    <!DOCTYPE html>
    <html lang="pt-BR">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Contador em JavaScript</title>
    </head>
    <body>

    <h1>Contador Simples</h1>
    <p id="contador">0</p>

    <button onclick="incrementar()">Incrementar</button>
    <button onclick="decrementar()">Decrementar</button>

    <script src="contador.js"></script>

    </body>
    </html>

Esse código cria uma estrutura simples com um título, um parágrafo onde o número do contador será exibido e dois botões: um para incrementar e outro para decrementar o contador.

Passo 3: Criar o arquivo JavaScript

Agora vamos criar o arquivo JavaScript que vai controlar o funcionamento do contador:

1. No mesmo diretório onde você salvou o arquivo HTML, crie um novo arquivo e salve-o com o nome "contador.js".
2. Dentro desse arquivo, adicione o seguinte código JavaScript: 

    
    // Inicializa o valor do contador
    let contador = 0;

    // Função para incrementar o valor
    function incrementar() {
    contador++;
    document.getElementById("contador").innerText = contador;
    }

    // Função para decrementar o valor
    function decrementar() {
    contador--;
    document.getElementById("contador").innerText = contador;
    }

Aqui está o que este código faz:

  • let contador = 0: Inicializa o contador com o valor 0.
  • incrementar(): Função que aumenta o valor do contador em 1 e atualiza o texto exibido na página.
  • decrementar(): Função que diminui o valor do contador em 1 e também atualiza o texto na página.

Passo 4: Testar o contador

Agora que você tem o código HTML e JavaScript prontos, vamos testar o contador:

1. Abra o arquivo contador.html em um navegador (Google Chrome, Firefox, etc.).
2. Você verá o número 0 no centro da tela, com dois botões: um para incrementar e outro para decrementar.
3. Clique nos botões e observe o número mudando conforme você interage com o contador.

Passo 5: Melhorias adicionais

Agora que o contador básico está funcionando, aqui estão algumas ideias para expandir o projeto:

- Adicionar um botão de "reset" para voltar o contador a 0.
- Definir limites para o contador, como não permitir valores negativos.
- Alterar o estilo do contador, mudando cores e tamanhos de fonte para personalizar a aparência.

Passo 1: Configurar o ambiente

Assim como no contador, você só precisará de um navegador e de um editor de texto (como Notepad, VSCode ou outro de sua escolha). Não há necessidade de instalar nada, pois o JavaScript é compatível com todos os navegadores modernos.

Passo 2: Criar o arquivo HTML

Vamos começar criando um arquivo HTML para ser a base do nosso trocador de cores. Siga os passos abaixo:

1. Abra seu editor de texto.
2. Crie um arquivo novo e salve-o com o nome "trocador_cores.html".

Adicione o seguinte código básico de HTML dentro do arquivo: 

    
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Trocador de Cores em JavaScript</title>
</head>
<body>

    <h1>Trocador de Cores</h1>

    <button onclick="trocarCor()">Trocar Cor</button>

    <script src="trocador_cores.js"></script>

</body>
</html>

Esse código cria uma página HTML simples com um título e um botão. O botão irá ativar a função JavaScript para trocar a cor de fundo.

Passo 3: Criar o arquivo JavaScript

Agora vamos criar o arquivo JavaScript que vai definir a função de trocar a cor de fundo. Siga os passos:

1. No mesmo diretório onde você salvou o arquivo HTML, crie um novo arquivo e salve-o com o nome "trocador_cores.js".
2. Adicione o seguinte código no arquivo JavaScript: 

    
function gerarCorAleatoria() {
    const letras = '0123456789ABCDEF';
    let cor = '#';
    for (let i = 0; i < 6; i++) {
        cor += letras[Math.floor(Math.random() * 16)];
    }
    return cor;
}

function trocarCor() {
    const corAleatoria = gerarCorAleatoria();
    document.body.style.backgroundColor = corAleatoria;
}

Explicando o código:

  • gerarCorAleatoria(): Gera uma cor hexadecimal aleatória (como "#FF5733").
  • trocarCor(): Muda a cor de fundo do body do documento para a cor gerada.

Passo 4: Testar o trocador de cores

Agora que temos o código HTML e JavaScript configurado, vamos testar o trocador de cores:

1. Salve ambos os arquivos (trocador_cores.html e trocador_cores.js).
2. Abra o arquivo trocador_cores.html em um navegador.
3. Clique no botão "Trocar Cor" e observe a cor de fundo mudando para uma cor aleatória sempre que clicar.

Passo 5: Melhorias adicionais

Agora que o trocador de cores básico está funcionando, aqui estão algumas ideias para expandir o projeto:

- Adicionar mais botões para escolher entre diferentes paletas de cores (por exemplo, cores pastel, cores vibrantes).
- Mostrar o código da cor atual em um elemento de texto na página.
- Definir um intervalo para que as cores mudem automaticamente a cada poucos segundos.

Pronto! Agora você tem um trocador de cores em JavaScript. Sinta-se à vontade para personalizá-lo e adicionar mais funcionalidades!
